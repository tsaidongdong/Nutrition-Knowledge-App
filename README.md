# Nutrition Knowledge Machine (營養知識機)

A Windows desktop application developed in **C#** that serves as an interactive educational platform for nutrition and health. Users can explore the relationship between food ingredients, nutritional intake, and their effects on different human body organs.

## 🏗️ System Architecture
The application follows a hierarchical navigation structure:

* **Main Menu:** The entry point to start the exploration.
* **Category Selection:** Choose between Ingredients, Nutrition, or Body Parts.
* **Detailed View:** Access in-depth data and specific nutritional facts.

## 🛠️ Technical Stack
* **Language:** C#
* **Framework:** .NET Framework / WinForms
* **IDE:** Visual Studio
* **Data Handling:** Local text-based database (`Food.txt`, `Normal.txt`)

## 📂 File Structure
* **Form1.cs to Form5.cs:** Handles different UI screens and logic.
* **Food.txt:** Stores ingredient-related data.
* **Normal.txt:** General nutrition or system information.
* **project.sln:** Visual Studio solution file.
